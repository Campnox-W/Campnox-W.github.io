## 算法 (Algorithm) 与模型 (Model) 的区别

在生物信息学和人工智能领域，**算法**和**模型**是两个紧密相关但截然不同的概念：

|          | **算法 (Algorithm)**                                      | **模型 (Model)**                                           |
| -------- | --------------------------------------------------------- | ---------------------------------------------------------- |
| **定义** | 解决问题的具体步骤或计算过程。                            | 从数据中学习到的参数化表示，用于预测或描述数据的内在规律。 |
| **作用** | 如何从数据中学习（**学习过程**）                           | 学到了什么（**学习结果**）                                  |
| **举例** | - 梯度下降法（用于优化）<br>- 动态规划（用于序列比对）    | - 线性回归参数 \(y = wx + b\)<br>- 隐马尔可夫模型的状态转移概率矩阵 |
| **生物信息学应用** | BLAST 的启发式搜索算法；比对中的 Needleman-Wunsch 算法 | 基因预测的 HMM 模型；蛋白质结构预测的 AlphaFold 神经网络模型 |
| **关系** | 算法用于训练模型（如梯度下降优化神经网络的权重），也用于使用模型进行预测（如用训练好的 HMM 解码序列）。 | 模型是算法在数据上运行后得到的产物，可独立用于推理。 |

总之，算法描述的是“**怎么做**”；模型描述的是“**做出来**”的参数化结果。在机器学习中，算法通过训练数据“学习”出模型，然后用模型对新数据进行预测。

## 本学期生物信息学学习计划

### 总体目标
- 掌握 Linux 基础操作与命令行环境下的数据处理
- 使用 R、Python语言进行统计分析与可视化
- 理解二代测序数据分析流程（RNA-seq、ChIP-seq 等）
- 初步掌握机器学习在组学数据中的应用

### 时间安排

| 时间段       | 学习内容                                                     | 实践任务                                                                 |
| ------------ | ------------------------------------------------------------ | ------------------------------------------------------------------------ |
| **第1-4周**  | Linux 基础：文件系统、权限、管道、正则表达式、Shell 脚本    | 完成 Linux 练习题；安装软件（如 FastQC、cutadapt）；搭建工作环境        |
| **第5-8周**  | R 语言入门：数据结构、数据操作（dplyr）、绘图（ggplot2）    | 完成 R 基础练习；尝试对一个小型表达矩阵进行探索性分析和绘图              |
| **第9-12周** | NGS 数据分析：RNA-seq 流程（比对、定量、差异表达）          | 使用 R/Bioconductor 包（如 DESeq2）分析公开数据；撰写分析报告            |
| **第13-16周**| 机器学习基础：监督学习（回归、分类）、无监督学习（聚类、PCA） | 用 R 或 Python（scikit-learn）对组学数据建模，比较不同模型性能           |
| **第17-18周**| 深度学习入门（选修）与课程项目                               | 参与小组经验分享；尝试用简单神经网络处理生物数据（如序列分类）           |

### 每周行动
- 课前预习：阅读教材对应章节，准备问题。
- 课堂参与：记录重点，积极提问。
- 课后复习：完成作业，整理笔记，与小组讨论。
- 周末实践：利用 GitHub 记录代码和笔记，尝试复现文献分析。
